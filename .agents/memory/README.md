# Memory Boundary

本目录只说明 Memory 边界，不存放真实 Memory 内容。

按照本项目的 Agent 持久化架构，Memory 用于保存代码和配置中看不出来、但对未来会话有帮助的观察，例如用户偏好、机器环境事实、外部资源指针或短期项目状态。这类内容通常是私有、动态或不可移植的，因此不应提交到本仓库。

## Do Not Commit

不要在本目录提交以下内容：

- API key、token、密码、邮箱凭据或其他敏感信息。
- 本机绝对路径、代理配置、私有账号或私有服务入口。
- 用户个人偏好。
- 当前会话进度、临时计划或短期项目目标。

## Where To Put Real Memory

如果所用 Agent 平台支持私有 Memory，应把真实 Memory 写入平台私有存储，而不是写入 `.agents/`。如果平台不支持 Memory，就在当前会话中临时使用，不要为了持久化而提交到仓库。
