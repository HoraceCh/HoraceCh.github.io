This note documents a cautious workflow for using AI tools during literature search and paper organization.

## Goal

Use AI tools to make searching, triage, and note organization more structured without outsourcing judgment or inventing sources.

## Workflow Draft

1. Write the research question in plain language.
2. Generate candidate Boolean search queries.
3. Search academic databases directly.
4. Check titles, abstracts, methods, and publication context.
5. Save useful papers with short reasons for inclusion.
6. Convert verified papers into reading notes.

## Guardrails

- Do not cite papers that have not been checked.
- Do not treat generated summaries as evidence.
- Keep the original search query and database source when possible.
