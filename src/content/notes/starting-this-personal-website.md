This is the first note for my personal website.

I want this site to be more than a static resume page. It will become a long-term public space for my learning notes, engineering projects, research reading, and reflections around mechanical engineering, AI-assisted workflows, VLA, embodied AI, robotics, and robot control.

## Why Build This Site?

When learning complex topics such as VLA, embodied AI, and robot control, useful information is often scattered across papers, code repositories, courses, blogs, and documentation. Without a stable place to organize the process, a lot of learning becomes fragmented.

The goals of this site are:

- Record ideas that I have actually studied and understood
- Organize paper reading notes and project reproduction work
- Show incremental project outcomes
- Build a learning system that can be updated over time

## What Will I Write About?

I plan to begin with:

1. VLA paper reading notes
2. Embodied AI concepts and learning paths
3. Robotics and robot control fundamentals
4. Project reproduction records
5. Personal learning methods and AI-assisted workflows
